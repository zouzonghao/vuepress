## 1. 包含'我爱你'


## 2. 包含大于'100 GB'的
```
\[(?:100(\.\d+)?|10[1-9]|[1-9][0-9]{2,}|[2-9]\d{3,})\sGB\]
```

^(?:\[日本AV\]|(?:\[.*?\]\s*)?(?:100(\.\d+)?|10[1-9]|[1-9][0-9]{2,}|[2-9]\d{3,})\sGB)
