## 2. 设置指定网段指定网关
```sh
route -p add 192.168.37.0 MASK 255.255.255.0 192.168.36.1 IF 10
```   
## 3、删除
```sh
route delete 192.168.37.0 MASK 255.255.255.0 192.168.36.1 IF 10
```
